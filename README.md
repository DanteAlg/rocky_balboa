# Rocky Balboa

## Requirements

- Ruby 2.0
- Portaudio >= 19
- Mpg123 >= 1.14


### OSX Install
```console
brew install portaudio
brew install mpg123

bundle install
```

## Get in action!

```console
FROM=2014-01-23 TO=2014-01-26 PROJECT=Project EMAIL=youremail@domain.com PASSWORD=password ruby rocky_balboa.rb
```
